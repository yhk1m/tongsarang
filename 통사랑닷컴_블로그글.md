# 통사랑닷컴 소개 및 공유 글입니다.



사회탐구 기출문제를 찾으려면 EBSi, 한국교육과정평가원, 시도교육청 홈페이지 등 여러 사이트를 돌아다녀야 합니다. 원하는 문항을 찾더라도 PDF 파일에서 일일이 스크롤하며 찾아야 하고, 단원별·난이도별로 분류하여 정리하는 것은 더욱 번거로운 작업입니다. 그래서 사회탐구 기출문제를 한 곳에서 검색하고, 필터링하고, 모의고사까지 만들 수 있는 프로그램의 필요성을 느끼게 되었습니다.

​

이러한 이유로 사회탐구 기출문제 서칭 프로그램인 통사랑닷컴을 개발하였습니다. 통사랑닷컴은 별도의 설치 없이 웹 브라우저에서 바로 접속하여 사용할 수 있으며, PC, 랩탑, 태블릿 환경 모두에서 동작합니다.

​

통사랑닷컴은 다음과 같은 의미가 내포되어 있습니다.

통합사회탐구를 사랑하는 사람들

통(通)하는 사랑 — 기출문제와 교육과정을 연결(통)하다

​

즉, 사회탐구 과목을 가르치고 배우는 모든 분들을 위한 기출문제 플랫폼이라는 의미입니다.

## 데이터 현황

통사랑닷컴에는 현재 **8개 과목**, 총 **9,101문항**의 기출문제가 탑재되어 있습니다.

​

| 과목 | 문항 수 | 이미지 수 | 시험 수 |
|------|--------|---------|--------|
| 한국지리 | 1,260 | 1,240 | 63 |
| 세계지리 | 1,220 | 1,220 | 61 |
| 통합사회 | 655 | 655 | 32 |
| 생활과윤리 | 1,220 | 1,220 | 61 |
| 윤리와사상 | 1,220 | 1,220 | 61 |
| 정치와법 | 1,178 | 1,178 | 61 |
| 경제 | 1,178 | 1,178 | 61 |
| 사회문화 | 1,170 | 1,170 | 61 |
| **합계** | **9,101** | **9,081** | |

​

수능, 6월/9월 모의평가, 학력평가 기출문제가 포함되어 있으며, 각 문항에는 정답, 배점, 정답률, 난이도, 발문, 문항 내용 등의 정보가 함께 제공됩니다.

한국사, 동아시아사, 세계사는 아직 데이터가 준비되지 않았으나 향후 추가할 예정입니다.

## 화면 구성

​

### 상단 — 과목 탭

​

상단에는 과목 탭이 나열되어 있습니다. 원하는 과목을 클릭하면 해당 과목의 기출문제 데이터가 로드됩니다. 현재 선택된 과목은 강조 표시되며, 과목 전환 시 필터가 자동으로 초기화됩니다.

​

---

​

### 필터 패널

​

통사랑닷컴의 핵심 기능 중 하나는 다양한 조건으로 문항을 검색할 수 있는 필터 패널입니다.

​

| 필터 | 설명 |
|------|------|
| **학년도** | 원하는 시험 연도를 선택할 수 있습니다. |
| **분류** | 수능, 6월 모의평가, 9월 모의평가, 학력평가 등 시험 종류를 선택할 수 있습니다. |
| **대단원** | 교과서 단원별로 문항을 필터링할 수 있습니다. |
| **성취기준** | 대단원을 선택하면 해당 단원의 성취기준이 동적으로 표시됩니다. |
| **난이도** | 상/중/하로 문항을 필터링할 수 있습니다. |
| **정답률** | 20% 이하부터 91~100%까지 9개 구간을 다중 선택할 수 있습니다. |
| **키워드 검색** | 발문이나 문항 내용에서 특정 키워드를 검색할 수 있습니다. |

​

한국지리, 세계지리, 통합사회의 경우 GeoTester 탑재 여부를 필터링할 수 있는 항목이 추가로 표시됩니다.

​

"필터 적용" 버튼을 누르면 조건에 맞는 문항만 테이블에 표시되며, "초기화" 버튼을 누르면 모든 필터가 해제됩니다.

​

​

### 문항 테이블

​

필터 결과는 테이블 형태로 표시됩니다. 각 행은 하나의 기출문항을 나타내며, 다음과 같은 정보가 컬럼으로 제공됩니다.

​

학년도 | 분류 | 번호 | 배점 | 답 | 대단원 | 성취기준 | 발문 | 문항 내용 | 정답률(%) | 난이도 | 문제보기

​

**분류 배지**: 시험 종류에 따라 다른 스타일로 표시됩니다. 수능(11월)은 검정 배경에 흰 글자, 6월/9월 모의평가는 회색 배경, 학력평가는 연한 회색으로 구분됩니다.

​

**난이도 표현**: 난이도 '상'은 검정색 굵은 글씨, '중'은 회색, '하'는 연한 회색으로 표현하여 시각적으로 직관적인 구분이 가능합니다.

​

**정답률 색상**: 정답률이 낮을수록 짙은 색으로, 높을수록 연한 색으로 표시되어 어려운 문항을 한눈에 파악할 수 있습니다.

​

**발문/문항 내용**: 텍스트가 긴 경우 3줄로 축약되며, "더보기" 버튼을 클릭하면 전체 내용을 확인할 수 있습니다.

​

**정답률(%) 정렬**: 정답률 컬럼 헤더를 클릭하면 오름차순(▲) → 내림차순(▼) → 원래 순서로 토글됩니다. 정답률이 낮은 문항부터 정렬하여 어려운 문제만 모아볼 수 있습니다.

​

---

​

### 문제보기 (이미지 모달)

​

테이블의 "문제보기" 버튼을 클릭하면 해당 문항의 원본 이미지를 모달 창에서 확인할 수 있습니다.

​

- 좌/우 화살표 버튼 또는 키보드 방향키로 이전/다음 문항으로 이동할 수 있습니다.
- 현재 위치가 "3/45"와 같은 형태로 표시되어 전체 문항 중 몇 번째를 보고 있는지 알 수 있습니다.
- 필터가 적용된 상태에서는 필터된 문항 목록 내에서만 이동합니다.
- ESC 키 또는 모달 바깥 영역을 클릭하면 닫힙니다.

​

### 모의고사 만들기

​

통사랑닷컴의 가장 실용적인 기능입니다. 필터를 통해 원하는 조건의 문항을 선택한 뒤, 이를 조합하여 모의고사 PDF를 자동으로 생성할 수 있습니다.

​

모의고사를 생성하면 두 개의 PDF 파일이 다운로드됩니다.

- **문제지 PDF**: A4 2단 레이아웃으로 문항 이미지가 배치됩니다. 원본 문제 번호는 자동으로 새 번호로 변경되며, 페이지 헤더에 "Mock Exam"과 페이지 번호가 표시됩니다.
- **정답표 PDF**: 각 문항의 번호, 정답, 배점, 출처(예: 2023년 수능 1번)가 표 형태로 정리됩니다.

​

예를 들어, 정답률 50% 이하인 한국지리 문항만 모아서 "킬러 문항 모의고사"를 만들거나, 특정 대단원의 문항만 모아서 "단원별 모의고사"를 만들 수 있습니다.

​

​

### 성취기준 연결하기

​

각 기출문항을 교육과정 성취기준과 1:1로 매핑하는 기능입니다. 이 기능은 비밀번호로 보호되어 있습니다.

​

- **좌측 패널**: 학년도, 분류, 연결 상태(전체/연결됨/미연결)로 필터링한 문항 목록이 표시됩니다. 연결된 문항은 ●, 미연결 문항은 ○으로 구분됩니다.
- **우측 패널**: 선택한 문항의 이미지와 함께 성취기준 트리가 표시됩니다. 영역별로 접기/펼치기가 가능하며, 성취기준을 선택하면 자동으로 다음 문항으로 이동합니다.
- **진행률 바**: 전체 문항 대비 연결 완료 비율이 실시간으로 표시됩니다.
- **내보내기/가져오기**: 작업 내용을 JSON 파일로 저장하고, 이후에 다시 불러와 이어서 작업할 수 있습니다.

​

성취기준 연결 작업이 완료되면, 메인 테이블의 성취기준 컬럼에 해당 문항의 성취기준 ID가 표시됩니다. 이를 통해 "이 성취기준에 해당하는 기출문항이 무엇인지" 역으로 검색하는 것도 가능합니다.

## UI 디자인

​

통사랑닷컴은 Apple/Stripe 톤의 그레이스케일 미니멀 UI를 적용하였습니다. 폰트는 Pretendard를 사용하였으며, 불필요한 색상 사용을 최소화하고 흑백 톤으로 정보의 위계를 표현하였습니다. 테이블에는 홀수행 흰색, 짝수행 연한 회색의 줄무늬 배경(zebra stripe)을 적용하여 가독성을 높였습니다.

## 기술 스택

​

통사랑닷컴은 Vite + Vanilla JS로 구축된 정적 사이트입니다. 별도의 데이터베이스 없이 JSON 파일로 데이터를 관리하며, GitHub Pages를 통해 배포됩니다. 문항 이미지는 EBSi PDF에서 자체 제작한 크롭 파이프라인(PyMuPDF + pdf2image)을 통해 추출하였고, 아티팩트 제거 후처리를 거쳐 깔끔한 이미지를 제공합니다.
