- 스프레드시트에 `연락처` 헤더 추가 + Apps Script `doPost` 수정 + 새 버전 배포 후 테스트 필요

### 방문자 카운터 (Today / Total) 구현
- **방식**: Google Apps Script + 스프레드시트 (방법 1 선택)
- **구현 계획**:
  1. 새 스프레드시트(또는 기존 시트에 새 시트 추가)에 날짜별 방문 수 기록
  2. Apps Script에 카운터 API 추가:
     - `doGet` 시 `?action=visit` → 오늘 날짜 행 카운트 +1, today/total 반환
     - 또는 별도 Apps Script 배포
  3. 프론트엔드: 페이지 로드 시 API 호출 → 헤더/푸터에 today/total 표시
- **주의**: 기존 오류 제보 Apps Script의 `doGet`은 제보 목록 반환에 사용 중 → 별도 Apps Script 배포 권장
