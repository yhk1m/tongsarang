### Day 1 — 2026.02.18 (프로젝트 생성 ~ 핵심 기능 완성)

#### 프로젝트 초기 설정
- Vite + Vanilla JS 프로젝트 생성
- Vercel 배포 설정 (GitHub 연동)
- 한국지리 1,240문항 + 이미지 기반으로 초기 버전 구축

#### 과목 데이터 추가
- **통합사회**: 655문항 (2018~2025, 32개 시험)
  - 2024~2025 데이터 추가, 월별 분류 적용
  - 정답/배점 데이터 추가 및 정확도 개선 (전 시험 50점 총합 검증)
- **생활과윤리**: 1,220문항 (61개 시험)
- **세계지리**: 1,220문항 (61개 시험)
- **윤리와사상**: 1,220문항 (61개 시험)

#### UI 디자인
- Apple/Stripe 톤의 grayscale minimal UI 적용
- GeoTester 칼럼 조건부 표시 (한국지리, 세계지리, 통합사회만)
- 사이트 타이틀 "통사랑(tongsarang) 사회탐구 기출문제"로 변경

#### 주요 기능 구현
- **모의고사 만들기**: 선택한 문항을 PDF로 생성하는 기능
- **성취기준 연결하기**: 문항별 성취기준 매핑 기능 (비밀번호 보호)
  - 8개 과목 성취기준 데이터 추가
  - 데이터 기반 동적 과목 결정

#### 이미지 크롭 파이프라인 구축
- `fix_bottom_cutoff.py`: PDF → 문항별 이미지 크롭 (PyMuPDF + pdf2image)
- `cleanup_crops.py`: 크롭 후 아티팩트 제거 (우측 구분선, 페이지 번호, 확인 사항 푸터)
- `build_data.py`: 크롭 이미지 → public/ 복사 + JSON 생성
- 이미지 오른쪽 구분선 제거 (2차 크롭)
- 하단 컨텐츠 잘림 수정 (전체 이미지 재추출)

#### 이미지 품질 보정
- 윤리와사상: 스캔 PDF(2021_05, 2025_11) 크롭 품질 개선
- 생활과윤리: 849개 이미지 보정 (구분선 제거 + 여백 트리밍)
- 한국지리: 2017학년도 수능 20문항 추가

### Day 2 — 2026.02.20 (신규 3과목 추가 + 품질 개선 + 배포 전환)

#### 전 과목 이미지 아티팩트 제거
- 전 과목 3,945개 이미지 크롭 아티팩트 일괄 제거

#### 신규 3개 과목 추가
- **정치와법** (pollaw): 1,178문항 — EBSi PDF 크롤링 → 크롭 → JSON
- **경제** (econ): 1,178문항
- **사회문화** (socul): 1,170문항
- EBSi 과목 코드: 정치와법=`s_bnj`, 경제=`s_eco`, 사회문화=`s_samun`

#### 이미지 크롭 품질 개선
- `fix_bottom_cutoff.py` 개선:
  - `[N~M]` 연결문항 감지: 다양한 틸데 문자(～ U+FF5E, ∼ U+223C, 〜 wave dash) 지원
  - 멀티스팬 텍스트 처리 (PDF에서 `[`, `9`, `～`, `10`, `]`가 별개 span인 경우)
  - 연결문항 공유 지문이 이전 문항에 포함되는 버그 수정
- `cleanup_crops.py` 개선:
  - JPEG 아티팩트 대응 우측 구분선 제거 (`has_no_text` 체크 추가)
  - Strategy 4: 코너 페이지번호 제거 (작은 갭)
- 정치와법 이미지 코드 매핑 수정 (`pol` → `pollaw`)

#### 통합사회 이미지 재크롭
- 32개 PDF 전체를 `fix_bottom_cutoff.py`로 재크롭
- 655개 이미지 중 199개 후처리 (구분선 9개, 하단 아티팩트 190개)
- 오른쪽 글씨 잘림 문제 해결

#### Vercel → GitHub Pages 전환
- GitHub Actions 워크플로우 (`deploy.yml`) 생성
- Vercel 프로젝트 삭제
- GitHub Pages 활성화 (gh CLI 설치 및 API 호출)
- Vite `base` 경로 `/tongsarang/`으로 설정

#### UI 개선
- **발문/문항내용**: 3줄 제한 + 더보기/접기 버튼
- **성취기준**: 더보기 버튼으로 전문 확인 가능
- **분류 칸**: 너비 70px → 105px (1.5배)
- **줄무늬 배경**: 홀수행 흰색, 짝수행 연한 회색 (zebra stripe)
- 대단원/중단원 10단계 음영 제거

## 이미지 크롭 파이프라인

```
EBSi PDF 다운로드 (download_pdfs.py)
        ↓
PDF → 문항별 PNG 크롭 (fix_bottom_cutoff.py --subject {과목} --copy)
        ↓
PNG → JPG 변환 + public/images/ 복사
        ↓
이미지 후처리 (cleanup_crops.py --apply --subject {과목})
  - 우측 구분선 제거
  - 페이지 번호 박스 제거
  - 확인 사항 푸터 제거
  - 여백 트리밍
        ↓
JSON 데이터 생성 (build_data.py 또는 수동)
```

### 이미지 파일 명명 규칙
- 형식: `{학년도}_{월}_{코드}_{번호}.jpg`
- 예시: `2026_11_korgeo_01.jpg` (2026학년도 수능 한국지리 1번)
